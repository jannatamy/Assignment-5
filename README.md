Ques: What is the difference between var, let, and const?
Answer: Difference between var, let and const are,
var: var is function scoped, we can re-declare and update it as many times as we want which sometimes leads us to bugs.
let: let is block scoped, we can update it's value but we can't re-declare it in the same scope.
const: It is short for constant. Once we give it a value, we can't change it or re declare it throughout our code.

Ques: What is the difference between map(), filter(), and forEach()?
Answer: The difference between map(), filter() and forEach() are,
map(): It runs a function on every item and returns a new array of the same length with the modified values.
filter(): It checks every item against a condition and returns a new array containing only the items that passed the test.
forEach(): It just runs a function for every item. It does not return any new array. It returns undefined.

Ques: What are template literals?
Answer: In JavaScript, Template Literals are a way to create strings using backticks (`) instead of single (' ') or double (" ") quotes.
By using this, we can write multi line strings very easily.
